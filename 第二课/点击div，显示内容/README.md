可以通过`children`和`getElementsByTagName`获取父元素下的子元素，`getElementsByTagName`获取同类型的子元素，`children`可以获得父元素下所有元素。

可以用`innerText`、`textContent`、`innerHTML`元素内部文本。
`innerText`和`textContent`获取的文本不包含内部元素，`innerHTML`包括元素标签。
